## Installation

You can install the package via composer:

```bash
composer require tmbenhura/jasper4php-cli
```

## Usage

```php
$skeleton = new Jasper4PHPCli\Jasper4PHPCli();
echo $skeleton->echoPhrase('Hello, Jasper4PHPCli!');
```
